# Normal Model

```
cw - change word
ciw - change inner word
viw - select inner word
vi' - select inner single quote
vi" - select inner double quote
ci' - change inner single quote
ci" - change inner double quote
vit - select inner tag(HTML Tag)
vat - select all tag(HTML Tag)
V - select line
u - undo
o - insert below line
O - insert above line
w - next word
b - previous word
gg - go to top
G - go to bottom
r - replace a single character
```
